GIS GPS Data Logger

Arduino based GPS data logger using external environmental sensors, outputting .csv file for import into ArcGIS.

Download this project as a .zip file Download this project as a tar.gz file

Welcome, and thank you for checking out my project!

What is this?

The overall goal of this project is to use ArcGIS and spatial datasets to solve a problem or, at the very least, output some useful conclusions.

With that goal in mind, this project quickly evolved into creating and using an Arduino-based GPS data logging device and a number of environmental sensors to collect data throughout downtown Chicago to be imported into ArcGIS for further exploration and analysis.

What's here?

This repository will be the home for my device designs, the code needed to get them working, and probably any notes and interesting output generated.

Author and Contributor

This project was started in 2013 and is currently maintained by David Ratliff (@1chicagodave). Questions, suggestions, and general feedback are welcome and appreciated.
